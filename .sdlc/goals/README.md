# Goals

One markdown file per goal. The loop runs them in filename order. Copy
`0001-example.md`, give it the next number, and fill the frontmatter:

- `done_when` — a checkable condition (a command/criteria), not a vibe.
- `lane` — `auto` lets the engine size it (small/medium/large).
- `status` — leave `pending`; the loop advances it to `done` or `parked`.
